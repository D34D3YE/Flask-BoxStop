Um die technische Funktionsweise der Web-Anwendung sicherzustellen, müssen einige Bibliotheken importiert werden, welche nicht mit Python installiert werden.

Die Installation der Bibliotheken wird durch das Ausführen der requirements.txt-Datei vereinfacht. 

Im Terminal können Sie folgenden Befehl ausführen:

pip install -r .\requirements.txt

Sollten die Bibliotheken bereits vorhanden sein, werden diese auf die neuste Version aktualisiert.

Nach erfolgreicher Installation der Bibliotheken kann die Web-Anwendung gestartet werden. 

Jetzt kann die main.py-Datei ausgeführt werden. Nach Ausführung der Datei sollte die Web-Anwendung nun über den Localhost gestartet werden.

Über folgende Adresse kann die Web-Anwendung nun aufgerufen werden:

http://127.0.0.1:5000
